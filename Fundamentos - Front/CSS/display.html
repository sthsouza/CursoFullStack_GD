elementos de bloco

São as tags que ocupam todo o espaço horizontal disonivel, possibilitam definir alturas e larguras ersonalizadas. O item sucessor de um elemento de bloco será mostrado em baixo do item.

Exemplo de elemento de bloco com tag p

divs, sections, p, elementos de ttulo (h1 até h6) são elementos de blocos.

Inline

Ao contário dos elementos de bloco, ocupa apenas o espaço necessário para o elemento. Não ocupando um linha horzontal toda. Vários elmentos inline podem ser renderizads juntos na página.Não 'possivel altrar a altura e nem a largura

Os spans, a tag "a", tag "em", "strong", "img" representam elementos que são inline

Display

Permite que um elemento de blocose torne um elemento inline e um elemento inline se torne um elemento de bloco

A tag div recebe valor inicial de block e a tag img recbe o valor de inline, mas existem outras vaariações

display: block; display: inline; display: inline-block; display: flex; display: grid;
Block
Faz com que oelemento de torne bloco, ele ocupará um linha inteira para ser rendeerizado (100% da largura disponivel)
inline
Faz com que qualuqer elemento se torne inline, ocuando aenas o espaço necessário para sua exbição. as propriedades hight e width não são processadas pela renderização.
Inline block
Torna possivel colocar um elemento do lado do outro (inline) ao mesmo tempo que mantém a capacidade de definir o weight e height (display)

flex
é usado para a disposição de elementos naas paginas, Com o flex os lementos passam a estar em um container que abriga outros elementos (flex itens), a organizaçã dos itens pode ser em coluna sou linhas. é usado para a criaçõa de layouts responsivos
Grid
asim como no flexbox, o grid também funciona como um container que te diversos ites. No entanto, trabalha d forma multidimnsional, no flex trabalhamos com colunas e linhas, mas no grid pdemos ter linhas colunas Combinados. O grid é mais para a constrçao de layouts, enquanto que o flex está mais para a disposição de elementos
