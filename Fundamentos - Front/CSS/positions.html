Positions

Refere-se ao posicionamento dos elementos em uma página html, a propridade position é usada para permitir que a posição dos elementos seja alterada.

position: relative;
position: absolute;
position: sticky;
position: fixed;

relative

Se baseia na posição static, é possivel usar as propiedades top, bottom, left e right. Indicam a distancia do lmnto apartitr da área definida. Caso nehuma propridade sej adefinida el terá o mesmo coportamento do static.

#square2 { position: relative; top: 20px; left 40px; }

Absolute

Permite que o elemento seja posicionado em qualquer lugar na tela, de forma livre, com as prpreidades Top, bottom, left right. O posicionamento inicial do absolute baseia-se num primeir elemento que não seja static. ou seja, Se o absolute esteja dentro de uma div com position relative a positions dele vai ser a partir dessa div

#square2 { position: absolute; top: 20px; left: 40px; }

Um elemento absolute deixa de ocupar seu espaço normal no documento e passa a flutuar com uma posição mais livre. Seu espaço no documento é liberado.

Fixed

acompanha o mesmo funcionamento da posição absolute, tem as mesmas propriedades. A bdiferença é que a absoluet baseia se no primeiro elemento antecessor não static equanto o fixed se baseia na tela em si

Ou seja, se a tela for movimentada, o elemento com position fixed continuará na tela enqunato o absolute terá seu elemento escondifo conforme a tela é movimntada.

#square2 { position: fixed;; top:20px; left: 60px; }

sticky

Tm as mesmas caracteristcas do relative e fixed, é be dinamica. enquanto sua posição não estiver visivel na tela, Sua posição será como um relative, ou seja, vai acompanhar a locomoção do html. mas assim que página for movida, ele vai fazer que nem o fixed. b>Só acompanha o movimento quanda a parte superior da tela passa sobre o elemento

As configrações são as mesmas dos outros elementos.

Overflow

Quando o coneteudo de uma página ultrapassa o limite máximo de uma página ou um uma div. é possivel ter o visible, hidden, scroll e auto

#content {
overflow: visible;
overflow: hidden;
overflow: scroll;
overflow: auto;

(tbm podemos definir por eixo)
overflow-y: auto;
overflow-x: hidden;
}

O comportaneto padrão dos elementos é o overflow visible, em que o conteudo ainda é visivel quando ultrapassa a fronteira de seu bloco.

o hidden é usado para esconder o elemento, o scroll adicona uma barra de rolagem para qu eo user possa ver o resto do conteudo. o auto só adiciona o scroll quando t5em o overflow;

flutuação

Elementos podem ser empurrados, seja pra esquerda ou direita. A propridade usada para esta função é o float que recebe os valores left, right ou none.

float: left;
float: right;
float: none;
}

O float ajuda na organzação de elementos sem modificar a popridade display. Assim, elementos de bloco podem sr posicionados lado a lado, de forma que flutuem para o mesmo lado.

#square1 {
float: left;
}
#square2 {
float: left;
}

No entanto, quando usmos o float, é como se estivessemos colocando os elementos para sobrepor os outros na página. elemento com float ficariam por cima de elementos com sm float.
