Pseudo-classes

Pseudo classes adicionam estiizações nos comportamnetos dos elementos, indentificando estados do elemento. O elemento pode representar diferentes estilos dependendo de seus estados.

Hover

Serve para mudar a cor do elemnto quando o mouse passa por cima.

div {
color: green;
}

div hover {
color: red;
}

No exemplo acima, a div está na cor verde mas quando o mouse ppassar por cima ela ficará na cr verde.

Pseudo-elementos

Os pseudo elementos permitem estilizar elementos já existemtes sem a necssidade de criar novos elemntos html. São como irmãos gemeos de um elemento, é possivel adcionar pseudo elementos antes e depois de um elemento.

Motivos

Criar elementos decorativos:
Adicionar linhas, ícones ou outros elementos visuais sem criar elementos HTML extras.
Simular elementos HTML:
Criar elementos como botões com pseudo-elementos, reduzindo o HTML.
Melhorar a semântica:
Manter o HTML mais limpo e semântico, evitando a criação de elementos div vazios para fins de estilização.
Criar layouts mais complexos:
Combinados com outras propriedades CSS, podem ser usados para criar layouts mais sofisticados.

Sintaxe

A sintaxe é formada por ::nome junto ao nome do elemento alvo. Por xemplo, para criar um botão com check box seria assim:

.button {
display: inline-block;
padding: 10px 20px;
background-color: #000
color: white;
txt-decoration:none;
border-radius: 4px;
}
.button::before {
content: "\2714"; /* Código Unicode para
um checkmark */
margin-right: 5px;
}

Outros exemplos

::before
Insere o conteúd antes do lemento, tipo um emoji antes de um texto em um botão.
::after
Mesma coisa que o before, só que depois da frase do botão
::first-letter
Permite estilizar a primeira letra de um elemento, como um parágrafo. Ideal para criar capitulares estilizadas;
::first-line
Permite estilizar a primeira linha de um elemento, como um parágrafo. Ótimo para destacar a primeira linha de um texto;
::selection
Permite estilizar o texto selecionado pelo usuário;
::marker
permit sstilizar os marcadores de listas
