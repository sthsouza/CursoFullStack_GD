Introdução ao HTML

O que é HTML?

A "Hyper text markup language" (linguagem de marcação de hiper texto) tem como principal objetivo criar a a estrutura básica de uma aplicação web ou site, funcionando como um esqueleto.

Principais tags de formatação

Tag bold bold destaca, deixando em negrito

Tag itálico itálico

Tag strong Bold mas com mais importância

Tag em Itálico mas com mais importancia

Tag mark marca o texto

Tag small deixa o texto pequeno

Tag del deixa o texto grifado

Tag ins sublinha o texto

Tag sub para formulás com números para cima

Tag sup para formulás com números para baixo
